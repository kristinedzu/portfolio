2gather

This is a single page web application, which contains a single
page architecture. In 2gather app user can browse through
unique events, sorted by dates, location and categories
as well as create and host new events. Creating a profile allows
user to keep his unique data and personalize the news feed.

Calendar function allows user to keep a track of the events
he signed up for or cancel the event. User is also able to buy
the tickets for paid events and store them in this app.
